# 🚀 My Landing Page

Landing page moderna e responsiva desenvolvida com **Next.js 14**, **Tailwind CSS** e **Framer Motion**.

## 📋 Pré-requisitos

- Node.js 18+
- npm ou yarn

## ⚙️ Instalação

```bash
# Clone o repositório
git clone https://github.com/seu-usuario/my-landing-page.git

# Entre na pasta
cd my-landing-page

# Instale as dependências
npm install

# Inicie o servidor de desenvolvimento
npm run dev
```
